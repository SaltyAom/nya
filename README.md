# Nyan
Collection of common components pattern for copy-pasting via CLI.

For example:
```bash
# This command will generate navbar
npx nya n navbar
```

## Meaning of the name
Nyan is an acronym of:
- N - collectio**N** of
- Y - common components pattern
- A - for copy-p**A**sting
